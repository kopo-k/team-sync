# TeamSync

チーム開発で「誰が何をしているか」をリアルタイムで共有するVS Code拡張機能です。

## 機能

- **GitHub認証**: GitHubアカウントでログイン
- **チーム作成・参加**: 招待コードでチームメンバーを招待
- **リアルタイム共有**: 編集中のファイルをチームメンバーと自動共有
- **ステータス設定**: 「ログイン機能実装中」などの作業内容を設定
- **同一ファイル警告**: 同じファイルを編集中のメンバーがいると通知

## 使い方

1. サイドバーの「TeamSync」アイコンをクリック
2. 「ログイン」ボタンでGitHub認証
3. チームを作成するか、招待コードでチームに参加
4. ファイルを編集すると、チームメンバーに自動で共有されます

## コマンド

- `TeamSync: ログイン` - GitHubでログイン
- `TeamSync: ログアウト` - ログアウト
- `TeamSync: チームを作成` - 新しいチームを作成
- `TeamSync: チームに参加` - 招待コードでチームに参加
- `TeamSync: ステータスを設定` - 作業内容を設定
- `TeamSync: チームを退出` - チームから退出
